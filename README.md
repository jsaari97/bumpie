# Usage
<!-- usage -->
```sh-session
$ npm install -g bumpie
$ bumpie COMMAND
running command...
$ bumpie (-v|--version|version)
bumpie/0.0.0 darwin-x64 node-v12.11.0
$ bumpie --help [COMMAND]
USAGE
  $ bumpie COMMAND
...
```
